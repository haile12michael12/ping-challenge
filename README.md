# Challenge 01: PHP 7.3 + gRPC Echo with React

## Description
A basic echo server using Spiral gRPC. React sends a string, PHP returns it.

## Requirements
- PHP 7.3
- Composer
- Node.js
- Docker (optional)

## Setup

### Backend
```bash
cd backend-php
composer install
php server.php
